# 3pScan

## 目标

· perfect  完美
· precise  精确
· Painless 无痛

## 使用
```
./3pScan -mode port,httpx -hostfile url.txt -top-ports 100

  -mode string
    	扫描模式 (default "port")
  -host string
    	主机地址
  -hostfile string
    	主机文件
  -ports string
    	端口
  -top-ports string
        常用端口，full,100,1000
  -ports-file string
    	端口文件
  -timeout int
    	超时时间,默认1秒 (default 1)
  -version
    	显示版本
```
