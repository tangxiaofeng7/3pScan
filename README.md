## Usage
```
# 简单扫描
./3pScan -h baidu.com -top 1000

# 文件读取方式扫描
./3pScan -hf url.txt -top 1000

# 全端口扫描
./3pScan -h baidu.com -top full
或
./3pScan -h baidu.com -p 1-65535

# 端口扫描进度提示，默认为5秒
./3pScan -h baidu.com -p 1-65535 -t 10

```
